# La Escoba — MVP & Implementation Plan

## Game Summary

- **Rules:** Standard "La Escoba" (Spanish 40-card deck, 2–4 players, play to 15 points)
- **Modes:** Single-player (vs. AI), Local multiplayer (pass-and-play)
- **Tech:** Angular (web)

## Implementation Epics

### 1. Game Engine Core

- Deck creation, shuffling, dealing
- Player entities, turn and scoring logic
- End-of-round and win condition

### 2. User Interface

- Game table, player hands, table cards
- Scoreboard, round summary, feedback
- Main menu, game setup (single/local multiplayer)

### 3. AI Opponent

- Basic logic for single-player mode

### 4. Local Multiplayer

- Pass-and-play support
- Player switching and state management

### 5. Polish & Testing

- Animations, sounds, visual feedback
- Unit tests for game logic
- Manual playtesting

**Next:** Start with Game Engine Core, then UI, then AI/local multiplayer, then polish and test.
